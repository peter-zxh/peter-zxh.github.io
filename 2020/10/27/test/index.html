python生成云词图

1.安装所需模块

pip install jieba 
pip install wordcloud
pip install matplotlib

2.准备

  • 歌词文本txt
  • 字体
  • 词云的背景图

3.上代码

import matplotlib.pyplot as plt
import jieba
from wordcloud import WordCloud

text = open('geci.txt', 'r', encoding='utf-8').read()
# 将歌词剪开
cut_text = jieba.cut(text)
# 以空格拼接
res = ' '.join(cut_text)
# 生成词云
wc = WordCloud(
    font_path='simkai.ttf',  # 字体
    background_color='white',
    width=1000,
    height=800,
    max_font_size=50,
    mask=plt.imread("4578.jpg"),  # 背景图片
    max_words=1000
)
wc.generate(res)
wc.to_file('res.png')
